# AI-Powered Reading Summarizer Web App

A **web app** that cleans up article pages and generates a **3-point AI summary**.


## Features

- Paste a URL to get a clean reading view.
- Generates AI-powered summaries using OpenAI GPT-4.
- Responsive design with Tailwind CSS.
- Modern, minimal UI for easy reading.



## Usage

1. Clone or download the repo:
```
git clone https://github.com/yourusername/ai-summarizer-web.git
```
